Languages Learnt:



Frontend:


HTML

HTML (HyperText Markup Language) is the standard language for structuring content on the web. It defines elements such as headings, paragraphs, images, and links, forming the foundation of any webpage.

CSS

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It controls visual aspects like colors, fonts, spacing, and positioning, allowing developers to create responsive and attractive web designs.


Backend:


Python

Python is a versatile, high-level programming language known for its simplicity and readability. It is widely used in web development, data science, artificial intelligence, and automation, making it a favorite among beginners and professionals alike.

Java

Java is a powerful, object-oriented programming language that is platform-independent due to its "Write Once, Run Anywhere" capability. It is commonly used for building enterprise applications, mobile apps (Android), and backend systems.



Hardware Programming


Arduino

Arduino does not strictly fall into either frontend or backend development. Instead, it belongs to embedded systems and hardware programming. It is primarily used for interfacing with physical components like sensors, motors, and LEDs, making it popular for IoT and robotics applications. However, it can interact with backend systems via APIs, databases, or cloud services.
